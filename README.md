
# Beauty by Briers — Gold & Lavender (Final)

Deploy-ready Next.js 14 + Tailwind project for a mobile hair & nail service serving care/nursing homes.

## Quick Start
npm i
npm run dev  # http://localhost:3000

## Deploy to Vercel
Push to GitHub → New Project → Import repo → Deploy.

## Customise
- Phone/email: components/Contact.jsx
- Services & prices: app/services/page.jsx
- Gallery placeholders: public/gallery*.svg
- Colours: tailwind.config.js & app/globals.css
